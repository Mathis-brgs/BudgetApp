# BudgetApp (MVP)

Application mobile React Native conçue pour la gestion de finances personnelles, avec un focus sur la simulation et la projection d'épargne à moyen/long terme. 

## Fonctionnalités du MVP

* **Comptes et Soldes :** Suivi manuel des comptes courants et des différents comptes d'épargne (Livret A, PEG, etc.).
* **Saisie Hebdomadaire :** Interface optimisée pour l'ajout manuel des transactions.
* **Simulateur d'Épargne Prévisionnelle :** * Création de plages d'épargne dynamiques (ex : +500 €/mois de mai à décembre, puis +300 €/mois).
    * Intégration de décaissements anticipés (ex : déduction de 850 € prévus pour des billets d'avion).
    * Visualisation de l'évolution du capital global dans le temps.

## Stack Technique

* **Frontend :** React Native (Framework Expo)
* **Backend / Base de données :** Supabase (PostgreSQL)
